# Notification Channels

## Notification Philosophy

Notifications should be rare. The decision inbox is the normal place for executive input. Text/chat alerts are for true urgency or explicitly whitelisted people.

## Suggested Channels

- Notion Inbox: normal decision queue
- Digest: low urgency summaries
- Chat: reply channel for mobile commands
- Text/iMessage/SMS: urgent only
- WhatsApp: optional urgent channel

## Recommended Ladder

1. Archive silently
2. Log or digest
3. Add to Notion inbox
4. Notify in chat
5. Text urgent channel

## Mobile Reply Pattern

Every urgent notification should include:

- Triage ID
- One-line decision brief
- Notion page link
- Reply instruction

Example:

```text
CEO@COMPANY-ALPHA
Urgent: domain renewal payment failed.
Open: https://notion.example/...
Reply here with the triage ID and what you want done.
```
