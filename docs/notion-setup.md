# Notion Setup

Create two databases.

## Rules Database

This is the editable policy layer. Use [templates/notion-rules-schema.md](../templates/notion-rules-schema.md).

Recommended views:

- By Outcome
- Active Rules
- Delegation Rules
- Urgent Notification Rules
- Needs Review

## Decision Inbox Database

This is the executive-facing queue. Use [templates/notion-inbox-schema.md](../templates/notion-inbox-schema.md).

Recommended views:

- Inbox
- Waiting For Executive
- Ready To Act
- Urgent
- Done

## Page Layout

Put the human-readable brief at the top of each decision item:

1. Last message date
2. People
3. Decision brief
4. Suggested reply/action
5. Context
6. Why it matters

Keep the original email link near the top of the property panel.
